# Projects

## 01. Solar PV-Battery System Optimization

### Objective
Develop an optimization model for sizing a solar PV and battery storage system capable of meeting the energy demand of an engineering workshop while minimizing lifecycle cost.

### Key Questions
- What is the optimal PV size?
- What battery capacity is required?
- How does system reliability change with different configurations?
- What is the economic feasibility of the system?

### Tools
Python | Excel | Power BI

### Dataset Sources
- NASA POWER
- Literature-based load profiles

### Status
🔄 Planned
